約数列挙・素因数分解 - 競プロ実装メモ

約数列挙

与えられた整数の約数を列挙します。

考え方

  1. NNN が与えられた場合、111 から N\sqrt{N}N​ までの整数 iii で試し割りをします
  2. NNN が iii で割り切れた場合、iii は約数です
  3. j=N÷ij = N \div ij=N÷i として、i≠ji \neq ji=j の場合、jjj も約数です

実装例

  • AtCoder Beginner Contest 180, C 問題
# 約数列挙
def factorize(n):
    res = []
    for i in range(1, int(n ** 0.5) + 1):
        if n % i == 0:
            res.append(i)
            j = n // i
            if i != j:
                res.append(j)
    return res


N = int(input())
print(*sorted(factorize(N)), sep="\n")

素因数分解

考え方

  1. NNN が与えられた場合、n=Nn = Nn=N として、nnn を 222 及び 333 以上の奇数 iii で割り続けます。
  2. nnn が iii で割り切れた場合、iii は NNN の素因数です。n=n÷in = n \div in=n÷i として再度同じ iii で試し割りをします。
  3. nnn が iii で割り切れない場合、次の数で試し割りをします。i2≤ni^2 \leq ni2≤n の範囲内でよいです。
  4. 最後に n≠1n \neq 1n=1 の場合は、nnn も素因数です。

実装例

  • AIZU ONLINE JUDGE, NTL_1_A
# 素因数分解
def prime_facotrize(n):
    res = []
    while n % 2 == 0:
        res.append(2)
        n //= 2
    i = 3
    while i ** 2 <= n:
        if n % i == 0:
            res.append(i)
            n //= i
        else:
            i += 2
    if n != 1:
        res.append(n)
    return res

N = int(input())
print("{}:".format(N), *prime_facotrize(N))
